Explicación Practica

Para la realizacion de la practica vamos a crear un repositorio en el cual vamos a subir nuestros archivos, seguidamente vamos a compartir este repositorio con nuestros colaboradores, además le vamos a otorgar permisos para la creacion y modificacion de dichos documentos.

Dicho esto lo primero que vamos a crear es nuestro repositorio de GitHub en el cual subiremos los archivos pertinentes, en este caso Index.html y su correspondiente hoja de estilos .css, para la creacion del repositorio deberemos ir a la aplicacion de GitHub Desktop deberemos ir al apratado de file, en el cual pincharemos sobre new repository, nos aparecera la siguiente pestaña en la que tendremos que poner el nombre de nuestro repositorio. Finalmente le daremos al boton de crear repositorio, con esto tendriamos nuestro repositorio creado.

El siguiente paso sería compartir el repositorio previamente creado con nuestros colaboradores, en este caso Rocio y Rafa, para ello deberemos irnos a nuestra cuenta de GitHub, concretamente en nuestro perfil donde encontraremos nuestros repositorios y seleccionaremos el que creamos anteriormente. Una vez hayamos seleccionado el repositorio iremos al apartado de settings/collaborators, una vez estemos dentro pincharemos sobre add people, seguidamente nos aparecera una pestaña en la que nos preguntara el nombre de usuario o mail de nuestro colaborador. Tras esto solo quedaria que el colaborador acepte la invitacion de acceso a nuestro repositorio.
